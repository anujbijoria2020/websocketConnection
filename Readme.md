## WebSocket Broadcast Console
A simple real-time WebSocket broadcast system built using Node.js and the ws library.
It allows multiple clients to connect to a server and broadcast messages to all connected users instantly.

## Features
- WebSocket server using ws
- Simple HTML client interface
- Real-time message broadcasting
- Live connection status indicator
- Live message logs with timestamps
- Multiple client support

## Tech stack
-Node.js
-ws (WebSocket library)
-HTML
-Vanilla JavaScript
